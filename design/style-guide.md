# Front-end Style Guide

## Layout

Les designs ont été créés dans les largeurs (widths) suivantes:

- Mobile: 480px
- Tablette: 768px
- Desktop: 1280px

## Colors

### Primary

- 050505  /  ebebeb

### Secondary

- 6f4a8e  /  221f3b 

## Typography

Heading : 24px/34px/50px/70px
Text: 12px/16px/18px

### Headings

- Family: Yusei Magic (https://fonts.google.com/specimen/Yusei+Magic?preview.text_type=custom)

- Weights: 700

### Body

- Family: Yusei Magic (https://fonts.google.com/specimen/Yusei+Magic?preview.text_type=custom)

- Weights: 400
